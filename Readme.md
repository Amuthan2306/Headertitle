#React-native-header-title

# INSTALLATION

A simple and fully customizable cross-platform React Native component that implements a header-title.

# Getting Started
Prerequisites:
"react":"18.2.0",
"react-native": "0.71.3",
"react-native-vector-icons": "^9.2.0",

# Install
npm i react-native-header-title
